复数

\[z=a+bi\]

复数的表示

坐标表示，分为实部\(a\)，虚部\(b\)： \[z=a+bi\] 三角形式，\(r\)表示模，\(\theta\)为幅角 \[z=r(\cos\theta + i\sin \theta)\] 指数形式 \[z=re^{i\theta}\] 欧拉公式 \[ e^{ix}=\cos x+i\sin x \]

共轭复数

\[z=a-bi\]

四则运算

\[ (a+bi) + (c+di) = (a+c) + (b+d)i \] \[ (a+bi)(c+di) = (ac-bd) + (bc+ad)i \] \[ (a+bi) / (c+di) = [(ac + bd) / (c^2 + d^2)] + [(bc - ad) / (c^2 + d^2)]i \]

四则运算的推论

\[ a \cdot (c+di) = ac + adi \] \[ {(a+bi)}^2 = (a^2 - b^2) + 2abi \]

模

\[ |a + bi| = \sqrt{a^2 + b^2} \] 模的平方 \[ {|a + bi|}^2 = a^2 + b^2 \]

离散傅里叶变换（DFT）

参考网上课件 \[ F[n] = \sum_{k=0}^{N-1} f[k] e^{-i \frac{2\pi}{N} n k} \quad (n=0:N-1) \]

逆变换： \[ f[k] = \frac{1}{N} \sum_{n=0}^{N-1} F[n] e^{+i \frac{2\pi}{N} n k} \] 对应的逆矩阵是原始（对称）矩阵的复共轭的\(\frac{1}{N}\)倍
